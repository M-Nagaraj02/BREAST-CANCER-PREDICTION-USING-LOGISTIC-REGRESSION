## Overview

This project predicts **breast cancer** using **Logistic Regression**, a supervised machine learning algorithm. It classifies tumors as **malignant (0)** or **benign (1)** using the **Wisconsin Breast Cancer Dataset**.

---

## Dataset

* Wisconsin Breast Cancer Dataset (scikit-learn)
* 30 numerical features
* Target labels: 0 (Malignant), 1 (Benign)

```python
from sklearn.datasets import load_breast_cancer

data = load_breast_cancer()
```

---

## Methodology

* Algorithm: Logistic Regression
* Data split: 80% training, 20% testing
* Basic preprocessing and feature handling

---

## Results

* Achieved **92% accuracy** on test data
* Confusion matrix and classification report used for evaluation
